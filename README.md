# My_IPTV

重点收集江西移动服务器的IPTV资源，别问为什么，问就是江西移动宽带。

### 文件命名结构
> aabbccc.m3u

aa = 服务器所在的网络运营商（cm = 移动; cu = 联通; ct = 电信; ce = 校园网）

bb = 服务器所在的省、自治区、直辖市（jx = 江西; 以此类推……）

ccc = 服务器提供商等备注信息

### 目录
|  | 文件名 | 地区网络 | 备注 |
|:------:|:------|:------|:------|
